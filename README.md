# Shopping list
Shopping list to keep track of your items in your basket

# Created with
  1. React
  2. Redux toolkit
  3. SCSS

# How to set up:
Watch for SCSS files:
```bash
 cd .\shopping-list\
```
```bash
sass --watch src/sass:src/css
```

Start the server:
```bash
 cd .\shopping-list\
```
```bash
npm start
```
